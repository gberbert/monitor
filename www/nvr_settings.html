Armazenamento Global

O diretório é gerenciado automaticamente pelo sistema (go2rtc_bin/storage) para garantir integridade.
A limpeza automática ocorre quando o espaço livre cai abaixo de 5GB OU quando o uso desta pasta excede o valor configurado acima.

Inteligência Artificial (Google Gemini)

Chave de API do Google Gemini para melhoria de imagem.
Modelo rápido parar ler a imagem e gerar o prompt (Ex: gemini-1.5-flash-latest).
Prompt enviado ao modelo de visão para descrever a cena antes de gerar a nova imagem. (Variável dinâmica)
Prompt utilizado para a etapa de validação da imagem gerada (Auditoria).
Modelo generativo para recriar a imagem (Ex: imagen-3, stable-diffusion).
